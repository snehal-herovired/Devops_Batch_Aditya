* difference between fetch and pull

	fetch and pull are related to obtaining updates from a remote repository.


fetch :
	 it is used to retrieves the data from a remote repository but does not merge it into the local repository. 
	It allows you to see what changes have been made and interact with a remote repository without affecting your local repository 



pull:
	it is used to retrieves the data from a remote repository and merge it into local repository.
