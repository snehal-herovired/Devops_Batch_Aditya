A continuous integration and continuous deployment (CI/CD) pipeline is a series of steps that must be performed 
in order to deliver a new version of software. CI/CD pipelines are a practice focused on improving software 
delivery throughout the software development life cycle via automation.

ROLE OF JENKINS:
================

Jenkins is an open-source automation tool used to build and test software projects.
The tool makes it more convenient for developers to integrate changes to the project. 
Jenkins achieves Continuous Integration with the help of plugins.
